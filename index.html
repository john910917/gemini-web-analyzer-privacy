隱私權政策

資料收集

本擴充功能會收集以下資料：

  • 使用者設定的 API 金鑰
  • 使用者選擇的介面語言偏好
  • 使用者截取的網頁畫面

資料使用方式

收集的資料使用方式如下：

  • API 金鑰：用於存取 Google Gemini API 服務
  • 語言偏好：用於提供對應語言的使用者介面
  • 截圖內容：僅用於傳送給 Gemini API 進行分析

資料儲存

所有資料都儲存在使用者的本地瀏覽器中：

  • 使用 chrome.storage.local API 儲存設定
  • 不會將資料上傳到任何伺服器（除了發送給 Gemini API 進行分析）
  • 截圖資料僅在分析時暫時使用，不會永久儲存

資料保護

我們採取以下措施保護您的資料：

  • 所有資料都存放在本地，不會上傳到第三方伺服器
  • 與 Gemini API 的通信使用 HTTPS 加密
  • 截圖資料在分析完成後立即銷毀
  • 不會收集任何個人識別資訊

使用者權利

使用者擁有以下權利：

  • 隨時可以清除所有儲存的設定
  • 可以選擇不使用截圖功能
  • 可以隨時移除擴充功能，這會刪除所有相關資料

隱私權政策更新

我們可能會不定期更新此隱私權政策。重大更新時會通知使用者。
